
# Chat em Tempo Real - Sistemas Distribuídos e Paralelos

## Descrição
Aplicação de chat em tempo real desenvolvida como projeto para a disciplina de Sistemas Distribuídos e Paralelos do curso de Engenharia Informática do ISPPU.

## Funcionalidades
- [x] Enviar mensagens para todos na sala (chat geral)
- [x] Enviar mensagens de forma privada
- [x] Indicador "usuário está digitando"
- [x] Lista de usuários online
- [x] Histórico de mensagens (geral e privado) armazenado em SQLite
- [x] Interface responsiva e intuitiva

## Tecnologias Utilizadas
- **Backend:** Node.js, Express.js, Socket.IO
- **Banco de Dados:** SQLite
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Bibliotecas:** Socket.IO (comunicação em tempo real)
